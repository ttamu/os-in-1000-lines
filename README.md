# OS in 1000 Lines

https://operating-system-in-1000-lines.vercel.app/ja/

## ファイル構成

```
.
├── disk/              # Filesystem contents
├── kernel.c           # Kernel implementation
├── kernel.h           # Kernel headers
├── kernel.ld          # Kernel linker script
├── shell.c            # Shell implementation
├── user.c             # User-mode runtime
├── user.h             # User-mode headers
├── user.ld            # User-mode linker script
├── common.c           # Common utilities
├── common.h           # Common headers
└── run.sh             # Build and run script
```

## ビルド・実行

```bash
./run.sh
```
